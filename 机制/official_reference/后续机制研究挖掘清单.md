# 后续机制研究挖掘清单

当前已经完成 8X6B / 9E6Y 的可视化和 distance-interface baseline。下一步应继续挖掘更多与机制有关的研究，而不是马上做 docking 或最终抗体生成。

## 1. 结构论文方向

需要系统查：

```text
PVRIG Nectin-2 structural basis
CD112R CD112 crystal structure
PVRIG PVRL2 double-lock-and-key
PVRIG CC' loop F-strand charged residues
```

目标：从论文正文/补充材料中提取作者认为关键的残基、突变实验、结合模式解释。

## 2. 临床/抗体机制方向

需要查：

```text
COM701 PVRIG mechanism epitope blocking
anti-PVRIG antibody blocks PVRIG-PVRL2
Tab5 CPA.7.021 PVRIG antibody epitope
HR-151 PVRIG VHH blocking
```

目标：区分“binding antibody”和“blocking antibody”，找出阻断活性是否与某些表位区域相关。

## 3. 专利表位映射方向

需要查：

```text
US10213505B2 anti-PVRIG antibodies S67 R95 I97
WO2021180205A1 PVRIG binding protein 151 VHH
PVRIG epitope mapping cynomolgus cross-reactivity
```

目标：把 S67/R95/I97 这类专利线索与结构坐标、物种差异、阻断实验联系起来。

## 5. 后续模型应该吸收什么

后续 AI/筛选模型不应该只学一个 docking score，而应至少包含：

```text
1. 是否覆盖 core PVRIG-PVRL2 interface；
2. 是否接近 R95 所在局部表面；
3. 是否避开只围绕 S67 的非阻断倾向；
4. 是否有足够空间位阻阻止 PVRL2 贴合；
5. 是否保持 VHH developability；
6. 是否与 HR-151 / Tab5 CDR 相似性过高。
```
