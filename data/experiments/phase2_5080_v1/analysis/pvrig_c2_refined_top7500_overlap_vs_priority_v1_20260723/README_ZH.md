# C2-refined Top7500 与当前 Priority Top7500 重合分析

## 比较口径

- 旧集合：当前正在执行 25,000-job Docking 的冻结 Priority Top7500。
- 新集合：`TOP7500_C2_REFINED.tsv`。
- “重复”优先指候选 ID、完整氨基酸序列及 `sequence_sha256` 三者完全一致。
- “新增”指相对于旧 Priority Top7500 不在旧集合中的候选；它们来自既有候选池，并不等同于刚生成的 de novo 序列。

## 主要结果

- 新集合内部：7,500 个候选 ID、完整序列和 SHA256 均唯一，内部精确重复为 0。
- 与旧集合精确重合：1,280 条（17.07%）。
- 相对旧集合新增：6,220 条（82.93%）。
- 旧集合被替换：6,220 条。
- 两个集合并集：13,720 条；Jaccard 为 9.33%。
- 高置信核心 4,468 条中：748 条与旧集合重合，3,720 条相对旧集合新增。

## CDR3 背景

- 新集合有 5,650 个唯一 CDR3。
- 1,955 条新榜单候选的精确 CDR3 曾在旧榜单出现；5,545 条的精确 CDR3 未在旧榜单出现。
- CDR3 相同不等于完整 VHH 序列重复。

## 当前 Docking 结果可复用范围

- 1,280 条精确重合候选可复用当前协议下的双受体 seed917 结果。
- 其中 583 条在旧版 4,000 条 repeat sentinel 中，可继续复用 seed1931 技术重复。
- 其中 126 条在旧版嵌套 1,000 条 sentinel 中，可继续复用 seed3253 技术重复。
- 其余 6,220 条相对旧榜单新增候选没有本轮旧 Top7500 的对应 Docking 作业。
